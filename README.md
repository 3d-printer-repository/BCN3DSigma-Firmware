# BCN3D_Sigma_Marlin
Firmware for the BCN3D Sigma Printer, based on Marlin

Features
--------------

- Dual independent X extruders.
- Autolevel, autocalibration of XYZ axis.
- TouchScreen support.
- Insert/Remove filament
- Refined firmware for better printing experience.
